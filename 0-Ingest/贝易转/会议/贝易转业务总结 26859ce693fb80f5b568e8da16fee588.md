# 贝易转业务总结

时间: 2025年9月8日 10:00

截止上周，贝易转产品已接入"一体三翼"全业务线，全年累计交易额达36亿元，实现业务降本423万元，产品收入361万元。最近一周的渗透率数据：惠居省心租18.76%、北京链家40.45%、深圳链家70%，城市链家正陆续启动运营。目前正探索向贝壳外客户推广，已与自如、麦田进行初步沟通。

主要洞察：

1. **产品营销方面：**产品推广初期，我们针对C端用户推出支付满减营销（控制比例不超过0.05%），有效引导用户习惯养成，使渗透率快速提升至约20%。随着用户习惯逐步形成，即使营销投入逐步降至零，产品渗透率仍未明显下降。
2. **差异化优势方面：**贝易转除了价格优势外，主要差异化优势来自于对业务场景的深度适配——如二手收佣场景的多轮转账、租赁场景的"一房一账号"、存管场景的"在线转账、秒级到账"等。这些功能不仅提升了业务效率，同时优化了用户体验。
3. **产品外拓方面：**基于上述认知，贝易转下阶段计划以租赁和经纪行业为外拓主攻方向，充分复用贝壳内的能力沉淀，在相似场景中确立竞争优势。
