# 微企付基本

调研对象：微企付BD、新网银行

1. BD团队规模：20+5
2. 对外服务商（代理）报价：0.1%，标准报价0.15%
3. 主推B2B经营性收款，二手佣金、装修等可以进，但租金认为是C2B，不做
4. 付款行：
    1. 借记卡接入17家，全部迁移为订单支付
    2. 信用卡支持4家，非主推，成本高，0.15%包不住。
5. 收款行：
    1. 微企付以平台身份与银行合作项目户，客户平台和商户以二级商户身份入驻。
    2. 收款行5家，包括浙商、交行、新网、上海和工行。其中浙商量最大
    3. 收款行成本0.05%左右
    4. 银行和微企付存在「双向分佣」，即：微企付向银行支付服务费，银行通过代理微企付产品，获得0.05%左右的费率差（标准费率0.15%-0.1%）
    5. 非直连模式，采用户名、金额及时间认款。
6. 合作：
    1. 支持去品牌接入，适用于补充贝易转的小程序场景
    2. 代理价格0.1%，可谈
