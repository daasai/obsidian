# 离替招聘

时间: 2025年8月4日 14:24

为支撑贝易转2025年“一体三翼”GTV100亿、降本1500万核心目标，及2026年贝外收入贡献占比30%的长期战略，需新增1名产品运营岗（外拓方向），负责外部客户深度需求挖掘、行业服务落地及定制化场景方案设计。

**招聘必要性**：「贝内服务+贝外输出」，产品人力瓶颈明显

当前外拓业务已启动，但面临深度非标准化场景的核心挑战：

- 需求高度非标：外部客户（如长租公寓、保租房企业）需定制化支付分账（如房源属地化分账至30+城市公司）、特殊流程（租金押金组合支付），需逐客户深度调研（单客户20+小时）、输出定制方案（含技术对接文档、运营手册）；
- 对接链路冗长：单客户决策链涉及财务（费率谈判）、运营（流程适配）、法务（合同审核）等多部门，平均对接周期45天，需专人全程跟进（含10+次跨部门会议、3轮方案修订）；
- 现有团队无冗余：贝易转7人团队中仅1名产品经理，当前聚焦内部场景（贝壳体系）运营，外拓相关工作已占用其40%精力（周均16小时），导致内部需求进度不够，AI仅能处理标准化工作（智能跟单、自动报告），无法替代客户沟通与方案设计。

**岗位核心收益：**支撑贝易转目标，ROI显著

- **加速外拓落地：**专职承接后，可缩短客户对接周期30%（从45天→32天），支撑2025年外拓核心客户（自如、全房通等）快速推进，以支撑2026年GTV300亿目标。
- **保障长期增长：**2026年，贝外收入占比30%目标需提前布局客户关系，该岗位可同步沉淀行业解决方案（如租赁行业产品），降低后续客户复制成本（单客户落地周期再降20%）；
- **经济效益明确**：按照产品拓展计划，今明两年的团队人均产值分别为150万和300万，待招岗位人力ROI达6:1。

恳请批准新增岗位，助力外拓战略落地，支撑贝易转长期增长目标。
