# 银联线上收银台

Owner: Xavier Ma

银联线上收银台是银联联合商业银行、[收单机构](https://zhida.zhihu.com/search?content_id=233935502&content_type=Article&match_order=1&q=%E6%94%B6%E5%8D%95%E6%9C%BA%E6%9E%84&zhida_source=entity)等行业各方，设计研发的移动端支付产品，支持商户移动端APP或者WAP网页中拉起[云闪付](https://zhida.zhihu.com/search?content_id=233935502&content_type=Article&match_order=1&q=%E4%BA%91%E9%97%AA%E4%BB%98&zhida_source=entity)APP、手机Pay、银行APP（云网版、网银版）等支付工具完成支付。可以为用户提供便捷多样的支付工具，提升支付体验。

![](https://pica.zhimg.com/v2-30bc8f6638cebe787028b98605f0663a_1440w.jpg)

**产品形式：**可直接拉起云闪付App + 30多家主流银行App + 手机pay（品牌手机自带的钱包）来支付

**支持平台：**暂时只支持App，以后会支持H5

**优点：**

- 支付交易[费率](https://zhida.zhihu.com/search?content_id=233935502&content_type=Article&match_order=1&q=%E8%B4%B9%E7%8E%87&zhida_source=entity)优惠，节省成本
- 用户支付体验流畅，有效替补
- 覆盖广泛用户群体，丰富选择
- 银联、银行经常有营销补贴活动，刺激消费
- 与微信、支付宝无关，不用担心其风控

**商户案例：**携程、麦当劳、12306、京东到家、本来生活、网易严选、[驴妈妈](https://zhida.zhihu.com/search?content_id=233935502&content_type=Article&match_order=1&q=%E9%A9%B4%E5%A6%88%E5%A6%88&zhida_source=entity)、云集、叮咚买菜、朴朴超市、[药师帮](https://zhida.zhihu.com/search?content_id=233935502&content_type=Article&match_order=1&q=%E8%8D%AF%E5%B8%88%E5%B8%AE&zhida_source=entity)等

**入网条件：**商户业务合规，最好日均交易额50万以上（差一点也可商议）。交易量越大，费率越优惠。收单机构需要递交多种材料给银联审核并提供技术支持，要收取一定[接入费](https://zhida.zhihu.com/search?content_id=233935502&content_type=Article&match_order=1&q=%E6%8E%A5%E5%85%A5%E8%B4%B9&zhida_source=entity)。

产品简介

商户APP集成银联线上收银台（通用版）SDK后，支持用户在商户APP的支付场景调用**云闪付APP**、**手机Pay**、**银行APP**、**境外银联卡支付**等支付方式完成支付。银联线上收银台为商户提供一站式支付场景解决方案，为用户提供便捷多样的支付方式，提升支付体验。本产品为原手机支付控件产品的升级。

产品模式

**1、标准模式（首次支付）**：在商户订单支付页面选择银联入口，支持拉起云闪付APP、手机pay、银行APP等多种支付方式，用户可在线上收银台页面选择需要的支付APP，并完成支付。

![image.png](https://open.unionpay.com/upload/tj/image/1651024121461013039.png)

**2、标准模式（非首次支付）**：根据商户的需求，可通过后台配置“记住用户上一次成功支付方式”并自动跳转至上一次支付的APP；如需更换支付APP，可以选择“使用其他支付方式”或者返回按钮，返回线上收银台页面。

![image.png](https://open.unionpay.com/upload/tj/image/1651024141699012591.png)

**3、APP独立入口模式**：银联支持商户APP前置收银台内的任一支付工具，即在商户收银台页面可以直接调起用户选择的支付工具（银行APP、云闪付APP、手机pay）并支付，无需用户二次选择。

应用场景

本产品主要应用于线上商户APP集成银联线上收银台（通用版）SDK的支付场景。集成后，用户可在商户收银台界面，选择“银联”支付，并选择具体支付工具完成支付。

目前支持的手机系统有：iOS、Android、鸿蒙NEXT（目前仅支持调用云闪付APP和华为Pay）

![](https://open.unionpay.com/tjweb/resources/images/new/list/detail_list.png)
