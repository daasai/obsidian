# 微企付资料

Owner: Xavier Ma

[微企付](https://zhida.zhihu.com/search?content_id=259905313&content_type=Article&match_order=1&q=%E5%BE%AE%E4%BC%81%E4%BB%98&zhida_source=entity)是[腾讯金融科技](https://zhida.zhihu.com/search?content_id=259905313&content_type=Article&match_order=1&q=%E8%85%BE%E8%AE%AF%E9%87%91%E8%9E%8D%E7%A7%91%E6%8A%80&zhida_source=entity)旗下专为中小微企业设计的大额转账支付产品，主要解决企业经营采购场景中的大额支付痛点（如高费率、低限额、对账繁琐等），通过聚合银行转账能力，支持单笔最高100万的支付额度，同时兼容个人银行卡和企业对公账户付款。

一、微企付的核心特点

1. 大额支付能力

- 个人储蓄卡单笔最高100万（如建行、农行等）；企业网银无限额。
- 手续费低于微信支付（传统费率0.6%），具体费率需业务洽谈。

2. 适用场景

- 适用于批发、订货、供应链采购等B2B场景，需日交易额超100万。
- 目前saas服务、b2b交易的客户有：易订货，订货宝，天财商龙，美菜，冻品云，DC农业等；快消餐饮有：红牛，牛栏山，正新鸡排，喜茶，霸王茶姬等；
    1. B2B平台与电商服务商
        - 电商平台、订货系统（如订货宝、采货侠、舟谱云管家）等需处理大额订单的B2B平台，依赖微企付实现订单流与资金流的实时匹配，避免人工对账繁琐和错账风险。例如，快消品供应链、农副产品批发等场景中，平台需支持单笔百万级转账，微企付的“钱单一体”功能可大幅提升交易效率。
    2. 连锁企业与经销商
        - 连锁品牌（如喜茶、美菜）及区域经销商常面临终端门店支付限额低、风控拦截等问题。微企付支持个人/企业银行卡直接转账，单笔最高100万元，且无需绑卡，解决门店收款难题。
    3. 中小微企业与个体工商户
        - 浦发银行深圳分行联合微企付推出的服务显示，中小微企业可通过该工具降低手续费成本（部分场景免手续费），并实现订单与资金流的透明化管理，减少资金损失风险。
    4. SaaS服务商与多商户平台
        - 需要分账管理、合规经营的平台（如美业、大健康类SaaS服务商），借助微企付的银行账户体系分账功能，可规避“二清”政策风险，满足监管要求。
- 支持微信小程序、公众号等入口，是目前除微信支付外唯一可在小程序中使用的大额支付方式。

3. 资金安全与合规

- 资金通过银行虚户存管（默认[上海银行](https://zhida.zhihu.com/search?content_id=259905313&content_type=Article&match_order=1&q=%E4%B8%8A%E6%B5%B7%E9%93%B6%E8%A1%8C&zhida_source=entity)），[D+1日结算](https://zhida.zhihu.com/search?content_id=259905313&content_type=Article&match_order=1&q=D%2B1%E6%97%A5%E7%BB%93%E7%AE%97&zhida_source=entity)至商户实体账户，规避二清风险。
- 支付流程自动关联订单信息，实现“钱单一体”，减少人工对账。

二、开通条件与流程

![](https://picx.zhimg.com/v2-6a8366fc08f8b69549717566e940e2fd_1440w.jpg)

开通步骤

1. 前期报备

- 由合作分公司向微企付业务侧（机构合作部）提交商户材料，包括：品牌名称、年GMV、门店规模、业务场景说明等。

2. 资料审核与入驻

- 审核通过后，商户通过邀请链接完善资料：上传营业执照、法人身份证、超级管理员信息、上海银行授权函。
- 对公账户需完成“小额打款验证”。

3. 签约与账户开通

- 超级管理员微信扫码签约，同步开通微企付产品权限。
- 费用结算户需额外上传合作协议（如分账场景），审核通过后方可使用。

4. 技术对接

- 目前仅支持小程序支付直连接入，其他方式（APP付、扫码付等）需联系机构合作部评估。

四、注意事项

- 结算时效：默认D+1自动提现，退款到账依赖商户资金池状态。
- 分账限制：费用结算（如物流分润）比例≤订单金额30%，且需线下计算。
- 退款规则：单笔交易最多退款50次，退款总额不可超原订单金额。
