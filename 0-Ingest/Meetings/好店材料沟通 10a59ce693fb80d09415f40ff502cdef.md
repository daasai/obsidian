# 好店材料沟通

在听取“开展社区周边商业体的好店项目”的提案汇报后，首先，会议肯定了贝壳金融团队在合适的时间成功申请金融牌照的贡献，为公司做了备灾计划！同时，**希望金融牌照能在我们一体三翼的大业务中找到应用场景并用好，这是我们真正有核心竞争力的地方。**其次，会议肯定了金融支付平台团队的创新激情，认为基于基础技术做增量业务并无不妥，如果真能做成，带来的商业价值、数据价值和社区链接价值都是很大的。

就当前汇报内容而言，项目组在回答一些核心问题的确定性上还不够，因此，**要求项目组先不开展试点，进一步通过充分、广泛、深入的调研，完善业务模式的核心逻辑后，再次上会汇报，届时决定下一步行动计划。**

**“能否成为一门好生意”的关键是能回答清楚“我们的业务能为客户提供什么不可替代的价值？”。**从需求侧看，商户在当下场景中有什么痛点、有多痛？有多大体量的商户有这些痛点？从供给侧看，当前已有很多类似机构，我们有什么核心竞争力可以取代他们？解决商户痛点所需的核心能力是否经纪人可以满足？我们原有支持大额低频交易的技术能力是否也可以支撑小额高频新业务的开展？**前序问题需要通过足量、切实的调研加以验证，至少触达几百个调研对象，避免以偏概全、把单点需求扩大化。**同时，基于调研应输出类似于贝壳八大赋能的“答客问/宣导材料”，不论是BD商户时说清楚服务和价值，还是培训宣导经纪人、增强经纪人对该业务的信心，都是非常必要的抓手。最后，由于金融业务的敏感性，需要提前做好由此可能带来的舆情风险评估和应对策略。

由此，向**【产研事务线-金融支付平台负责人 宋博士】**提出待办要求：项目组对足量的商户展开深入且广泛的痛点及需求调研，明确业务提供给客户的不可替代的价值，再次完善业务核心逻辑后，可在后续小班委会议时再次提案汇报。
