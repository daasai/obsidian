# 微企付接入成本评估

微信微企付的接入，快速评估了一下，整体时间预估：8~9周

1. 前置沟通准备：（2周）
    1. 财务和风控准入：贝壳侧支付主体在上海银行开户、预充手续费等，需提交CFO线决策可行性；腾讯侧评估贝壳支付代付商户手续费的可行性。
    2. 产品合作方案：产品方案的细化沟通，包括用户端交互、订单认款、结算账户、对账等；
2. 产研流程：（2~3周）
    1. 收银台交互设计   2天
    2. 支付系统升级，包括收银台加载新支付工具、微企付通道接口对接联调 1~2周
    3. 链路测试联调  1周
3. 业财适配（2周）
    1. FSSC的对账单定制，微企付的资金结算直接到商户银行，账户对账单需要定制改造  2周
    2. （并行）理房通业财（收入/成本模块）改造，适配腾讯手续费方案 1周
4. 业务对接（2周）
    1. 业务培训 1周
    2. 业务试单及UAT 1周
