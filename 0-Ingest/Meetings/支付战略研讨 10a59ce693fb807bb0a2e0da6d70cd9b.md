# 支付战略研讨

贝易转：

产品体验打磨，以及一体三翼拓展；

价值：优化用户大额支付体验、降低业务支付费率成本；

资源支持：业务接入；

沉淀：大额转账产品、线上/线下支付产品运营策略；

银行账户：

银行账户标准化，拓展银行账户在收付、存管、分账、余额增值等领域的应用；

价值：体系化的解决银行账户合作方面，在系统交付效率、风控合规、资金安全、团队组织等方面的问题；

资源支持：机构合作、资源投入；

沉淀：银行账户的系统能力、合规方案、资金安全监控防护能力

好店：

服务社区商户的聚合支付方案；

价值: 服务商户降本提质、服务消费者更好的社区服务体验、帮助门店经纪人、拓展平台数据、商机和增长；

资源支持：人店资源支持、好店项目团队组建

智能一体机：

支付智能设备在整装大店、集签中心的落地；

价值：优化大店场所内的支付体验

沉淀：支付智能硬件的研发和设计能力
