# 整装广州政府补贴方案

1. 背景：广州政府以旧换新补贴，规模15亿
2. 广州：C端领取优惠券，B端通过政府POS 的B扫C完成收款，银商将实付资金结算给广州公司，广州公司凭POS订单号和发票找政府报销补贴费用
3. 异地：北京、武汉，线上支付云闪付小程序。
4. 改造范围：
    1. 广州圣都申请商编、终端，通道费不收
    2. 退款，净额结算？
    3. 新政府补贴接口：支付中台、收银台
    4. 收银台调起云闪付接口，云闪付按商编和商品ID识别补贴交易
    5. 业务回调，需要透传银商系统的订单好
    6. 业财维持现状：对账单不体现，银企直连银行流水与业务台账核对。
5. 技术方案选择：
    1. 收银台直接对接通道模式 【选择】
    2. 收银台对接核心交易链路模式
