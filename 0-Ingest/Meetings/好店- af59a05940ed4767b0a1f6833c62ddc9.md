# 市场情况总结：

- **需求维度（每家推荐产品看门店实际情况、码牌、云音箱、插件软件、POS、收银一体机）**
    1. 收付款需求商户：最直接需求就是费率，有营销活动、持续时间。
    2. 经营需求商户：比现有收款费率要低一点，更换方便，重要的是结合这家店描述出他的使用场景。出问题要可以找到人。成本设备便宜（鱼店例子：显示屏+软件+电子秤），
- **成功转化商户的特点：**
    1. 费率。一定要比现有费率低。
    2. 体验。有码牌和收银设备的不能比之前差，换设备不能影响现有操作习惯；没有设备的和他描述如何使用；
    3. 品牌背书：链家门店就在附近。
    4. BD讲解话术：客户问的问题要能回答上。和其他产品优势要明确。
- **商户资源：**
    1. 门店搬家过。当前位置周边商户数量很多，但是敢陌拜的经纪人极少。基本就是“吃老本”
    2. 对商户有意向的定义？金控是不拒绝就属于有意向，我的理解是能约好时间，可以下次再拜访就算。
    3. 带看集中在上午成单，下午是带看数量多但是未成单。上午更多是提前沟通过，实际去门店已经属于“二次沟通”。
    4. 下家门店会和店长提前沟通意向，判断是否要每位经纪人先定的1个客户的指标？

# 成功实践总结：

- 拓客经验：
1、当前拓的产品都是以云音箱居多。
--经纪人：
1）**提前准备**。在不了解产品时，足够熟络是关键。可复制经验是：了解产品后提前和熟悉的商户沟通一下，次日再和BD带产品去现场拜访，主要提及产品费率。
2）**不刻意介绍金控BD**，就是贝壳专业负责这个产品的部分同事，来推产品。
- -金控：
1）**商户就是有需求的**，比如文具店，建行的码，只支持储蓄卡，不支持花呗和信用卡；支付宝的如意POS，对于小孩儿的智能手表，无法刷码等等，需求需要和商户深聊就能带出来。
2）**想象和链家合作的画面**：先是周围熟人拓成功之后，再通过拓成功门店进行转介绍。
3）**带牌是真管用**。周边对贝壳的熟悉度是足够的，可以直接进店聊。
- -贝壳
    1. **拓客周期不定。**一家商户拜访多次属于正常，客户需求需要多挖掘，每家有差异，但是都有共性。后续可以针对行业+经营强度制定需求SOP
    2. **有无经纪人的区别？**
        - **人的熟悉**；
        - **品牌熟悉**，拓客的一条话术是门店就在这儿，而且如果金控带着链家的牌，同一家商户二次沟通，也没问题；
        - **拓不成时**，会有客户反馈有意向来门店沟通；

总结：

1. 冷启动：新业务处于启动的最早期，圈经、店东、经纪人还很难看到长期的收益（包括货币还是非货币收益）。当前阶段贝壳币是必要的方法，同时：
    1. 结合标杆示范带动其他人更多、更好的参与。
    2. 另外，带看和成单的激励额度差需要再扩大，引导经纪人以成功拓店为目标。
2. 现代金控：安排了最好的BD，赵壁总牵头，看好长期合作，配合程度很高，开局也不错。但有几个待改善的点：
    1. 金控BD和贝壳人间的沟通还没有完全打开，不利于运营SOP的快速打磨。
    2. 只有彼此持续提“要求”，才是好的开始。
3. 产品：尚处于早期，暂时谈不上产品优势，但不能比友商差，达到行业中上水平；费率是可以打的牌，可以做为敲门砖。但进门后还要把“熟人”、“就近”、“及时”的牌打出来，而且需要思考如何把这几张牌做好、表达好。
4. 商户：首先是数量，经纪人与周边商户的连接是真实存在、确定无疑的。但相对好店的预期规模还不足够，比如雍华府门店，22个经纪人的门店，当天可以连接的商户26家。原因：
    1. 经纪人缺少“有意识”的做过“连接”，有了业务就会有意识的去做。
    2. 基于第一波拓客基础，可以启动“转介绍”、“社区会员”、“大店/连锁直销”等进阶运营策略
