# 本周收获与反思

自去年以来，团队一直希望把有限资源用到刀刃上，聚焦核心目标。过程中遇到不少同事的疑问，也有一些思考，这里试着做下总结：

1. 聚焦就是拒绝“胡子眉毛一起抓”：我们面临的挑战往往有一定的复杂性，需要做的事情有很多，需要按20/80原则，找出当下最有效果、投产比最高的事情，集中投入资源做出效果。
2. 聚集是保持“行动力”：我们需要将“行动”设定为自己的默认模式，时间不应该用于做一个完美计划，而是带着一个“不完美”的计划果断前进，在行动中不断完善计划、修正动作、迭代认知。
3. 聚焦目标与全面认知并不矛盾，而且很多时候是相辅相成的：你需要在聚焦目标前进、确保不偏离的前提下，发现可能的机遇、变化和风险；
4. 一个组织/系统/复杂能力，随着不断的进化和发展，往往会成为由多种互相作用要素构成的“复杂系统”。面对“复杂系统”，我们唯一可行的做法就是保持“行动”，聚焦瞄准主要矛盾做出改变，而非试图控制“复杂系统”全局。
