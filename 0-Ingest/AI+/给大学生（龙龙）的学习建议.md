🚀 方向一：Vibe Coding（氛围编程/大模型驱动开发）

**核心逻辑：** 开发者不再从零编写代码，而是通过自然语言描述需求（Vibe），利用 AI 原生 IDE 自动生成、调试和部署应用。

1. 必用工具链（前沿阵地）

- **Cursor / Trae / Windsurf**：目前最强的 AI 原生 IDE。推荐重点练习 **Cursor** 的 `Composer` 模式和 **Trae** 的全库索引能力。
- **v0.dev / Bolt.new**：前端“一句话原型”工具。只需描述 UI，直接生成可运行的 React/Next.js 代码。
- **Replit Agent**：支持从零构建包含后端和数据库的全栈应用，并一键部署到云端。

2. 学习资源与实战指导

- **《Cursor 进阶指南》**：学习如何通过 `.cursorrules` 文件约束 AI 行为，实现企业级代码规范。
- **Prompt Engineering for Developers**：推荐吴恩达与 OpenAI 合作的最新系列微课，重点学习“结构化提示词”编写。
- **实战项目建议**：
    - _初级：_ 用 v0.dev 生成一个个人作品集网页。
    - _进阶：_ 使用 Cursor 配合 Claude-3.5-Sonnet 模型，在 2 小时内开发一个 Chrome 插件或 Todo 清单应用。

🤖 方向二：Agent 开发（自主智能体）

**核心逻辑：** 赋予大模型“手脚”（工具调用）和“大脑”（逻辑推理/规划），使其能够自主完成复杂的多步任务。

1. 核心框架与平台

- **LangGraph**：目前工业界最推崇的 Agent 框架，支持复杂的循环逻辑和状态保存（比起传统的 LangChain 更适合生产环境）。
- **Dify / Coze (扣子)**：零代码/低代码 Agent 开发平台。适合大学生快速上手，理解 Agent 的工作流（Workflow）、插件调用和知识库（RAG）。
- **CrewAI / AutoGen**：多智能体协作框架。学习如何让“程序员 Agent”和“测试员 Agent”自动协作完成任务。

2. 深度学习资源

- **Datawhale  Hello-Agent 课程**：国内最适合大学生的开源 Agent 入门教程，涵盖从原理到动手实现。
- **DeepLearning.ai《AI Agents in LangGraph》**：由 LangChain 创始人亲授，讲解如何构建具备自我纠错能力的智能体。
- **实战项目建议**：
    - _初级：_ 在 Coze 上搭建一个能够自动抓取学术论文并总结成日报的“科研助手”。
    - _进阶：_ 使用 LangGraph 构建一个具有“长期记忆”和“多步搜索”能力的本地法律/校园政策咨询 Agent。

---

🛠️ 跨方向必备：AI 基建与软技能

- **RAG (检索增强生成)**：学习如何将 PDF/数据库资料喂给 AI，解决大模型胡说八道的问题（重点看 **Learn RAG from Scratch** 开源库）。
- **Function Calling (函数调用)**：这是 Agent 的灵魂，必须掌握如何让大模型精准输出 JSON 格式来驱动外部 API。
- **API 调度技巧**：学会使用 **DeepSeek**（高性价比）、**Claude 3.5**（代码能力最强）和 **GPT-4o**（多模态最强）的组合方案。

---

📈 建议学习路径

1. **第一周 (感受 Vibe)**：下载 Cursor，尝试在不手动写一行代码的情况下，用对话方式改写一个小游戏（如贪吃蛇或 2048）。
2. **第二周 (掌握工作流)**：在 Dify 或 Coze 上配置一个带外部 API 调用的 Workflow。
3. **第三周 (深入代码)**：学习 Python 异步编程，尝试用 LangGraph 写一个多步骤的自动化脚本。
4. **持续追踪**：关注 GitHub 上的 **Trending** 榜单和 **Twitter(X)** 上的 AI 开发者社区（如 Andrej Karpathy 的分享）。

**总结建议：** 2026 年的学习不再是背诵 API，而是**“定义问题的能力”**和**“编排 AI 的能力”**。多动手、多 Prompt、多看源码。
