### 📐 布局解构 (Anatomy of the UI)

#### 1. The Sidebar (极窄侧边栏)

- **宽度**: 56px (收起状态)。
    
- **元素**: 只有图标，无文字。
    
    - `Grid` (Dashboard)
        
    - `Layers` (Strategies)
        
    - `Activity` (Logs/History)
        
    - `User` (Account)
        
- **视觉**: 半透明磨砂背景，无右边框 (No border-right)，与主背景融为一体。
    

#### 2. The Header (面包屑导航)

- **路径**: `My Strategies / 策略名 / 版本号`。
    
- **操作**: 右上角只有极简的文本按钮 `[Export]`, `[Deploy]`。
    
- **字体**: 使用 `Geist Sans`，灰色字，只有当前层级是白色。
    

#### 3. Left Panel: Logic Stream (逻辑流)

- **形态**: 时间轴 (Timeline)。
    
- **GenUI Card (双线框部分)**:
    
    - 这是 AI 生成的 **React 组件**，不是文本。
        
    - 包含 **Slider (滑块)** 和 **Tags (标签)**。
        
    - 按钮是高亮的（Primary Color），诱导点击。
        

#### 4. Right Panel: Alpha Canvas (画布)

- **HUD**: 巨大的等宽数字 (Monospace Numbers)，颜色标识涨跌。
    
- **Chart**: 占据最大面积，无边框，极细网格线。
    
- **Table**: 紧凑型表格 (Compact Table)，行高极小，信息密度极大。
    

#### 5. Bottom: Command Bar (指令栏)

- **位置**: 悬浮在屏幕底部中央，层级最高 (Z-Index: 999)。
    
- **质感**: 完全的玻璃拟态 (Glassmorphism)，带很深的阴影。
    
- **交互**: 按 `/` 键呼出，输入即执行。


我希望给它更多灵动的感觉

用户有几个关键时刻：

1. 当用户通过我们内容分享，第一次进入Alpha时，他看到的是：
	a.左侧一个正常比例的sidebar，菜单包括：新建叙事、我关注的（即:收藏叙事）、热门叙事，以及下方我的叙事（保存用户自己创建的叙事或是在预置叙事上做过调整的叙事，可能有多个）、用户设置；
	b.右侧：一个大大的chatbot input悬浮在底部；它的上面滚动着最近热门叙事卡片，每个卡片上都有一条回测/实盘价格曲线与基准价格曲线形成对比；叙事卡片流再上方是产品的演示动画：产品从市场和金融门户抓取海量的数据和事件，然后AI分析和蒸馏这些数据，形成各种各样的因子和因子组合；然后来自世界各地的金融和领域专家参与其中（human in the loop），分析这些数据和事件，加入自己的分析洞察，形成一条条价值千金的叙事，并持续在市场中释放可观的盈利。

2. 当用户点击某个叙事或者自己在chatbot中输入自定义叙事时，UI进入一段优雅的动画：chatbot上方的卡片流和动画消失，取而代之的对话逻辑流（LOGIC STREAM (GenUI) ），在逻辑流上方显示The Header (面包屑导航)。注意：这时右侧只有对话逻辑流，还没有ALPHA CANVAS (Visualization)  。
	- 当AI首先通过对话澄清用户对叙事，然后根据叙事生成一个「叙事控制卡片」，同时叙事画布（ALPHA CANVAS ）缓缓展开，显示骨架屏过渡后，展示回测/实盘曲线、绩效卡片和个股table
	- 这时，用户随时可以调整「叙事控制卡片」，也通过chatbot自然语言修正想法。而叙事画布随着这些动作动态变化。
	- 系统会自动保持叙事的定义，并自动出现在sidebar我的叙事列表；
	- 用户也可以点击「关注」，添加到sidebar 我关注的列表中。
	- 用户也可以选择叙事画布下方的股票，加入到自选股票中。，添加到「自选股票」

3. 当用户点击热门叙事或收藏叙事，右侧切换为叙事卡片画册，点击某个叙事卡片，展开成与2相同的画面。
4. 当用户点击自选股票菜单时，右侧展示自选股票清单表格。（这部分还没想好）
