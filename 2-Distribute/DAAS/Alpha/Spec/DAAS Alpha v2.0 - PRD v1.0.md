

# 📑 DAAS Alpha v2.0 产品需求文档 (PRD)

**最后更新**: 2026-01-29

## 1. 战略基石：第一性原理 (The Why)

我们不与量化巨头拼速度（高频），也不与公募基金拼深度（人海战术）。个人的优势在于 **“灵活性”** 与 **“技术红利”**。

- **赚谁的钱？**
    
    - 赚 **“情绪韭菜”** 的钱：利用散户的追涨杀跌与过度反应。
        
    - 赚 **“机构笨重”** 的钱：利用大资金调仓慢、风控死板的劣势，做它们“看不上”或“来不及”的机会。
        
- **凭什么赚？（不对称优势）**
    
    - **信息平权 (AI Leverage)**：利用 LLM (DeepSeek/GPT) 瞬间处理海量非结构化信息（研报、公告），在这个维度上，个人第一次拥有了与机构持平的信息处理能力。
        
    - **制度套利 (Regulatory Arbitrage)**：机构必须持仓，个人可以空仓。**“空仓”** 是个人投资者最大的特权（Alpha）。
        
    - **敏捷性 (Agility)**：船小好调头，专注于中短期的**基本面动量 (Fundamental Momentum)**。

## 2. 产品定义 (Product Definition)

**DAAS Alpha v2.0 到底是什么？**

它不是一个看行情的工具（如通达信），也不是一个被动的定投机器（如理财魔方）。

**它是用户的“AI 投资代理人” (AI Investment Agent)。**

它是一个 **7x24 小时工作的数字员工**，兼具以下三个角色：

1. **量化研究员 (Quant Researcher)**：用冷酷的数学（RPS/RSRS）筛选出全市场最强的标的。
    
2. **AI 分析师 (AI Analyst)**：阅读全网信息，为人提供“为什么买”的强逻辑解释。
    
3. **风控官 (Risk Officer)**：在用户上头时泼冷水，强制执行止损和仓位控制。
    

**核心特征**：

- **White-box (白盒化)**：拒绝“黑盒策略”，提供清晰的赚钱逻辑。
    
- **Action-oriented (行动导向)**：不堆砌数据，只提供“买/卖/空仓”的行动建议。
    
- **Defensive (防御优先)**：先不输，再求赢。

## 3. 目标用户与心理画像 (User & Psychology)

**目标用户**：有资产积累、渴望参与 A 股博弈，但受限于**时间**和**执行力**的个人投资者。

**核心矛盾**：

> **“既要不明觉厉的理论背书，又要一分钟看懂的傻瓜结论。”**

**解决方案：透明的黑盒 (The Transparent Box)**

- **不明觉厉**：底层必须是硬核的数学模型（RPS 99, RSRS 0.8, 乖离率），这是建立**信任**的基石。
    
- **一分钟看懂**：顶层输出必须是**“自然语言逻辑”**。用户不需要看公式，只需要看到：“这只票主力资金连续流入（RPS强），且踩中了‘低空经济’风口（AI确认），建议跟进。”

## 4. 核心策略体系 (The Alpha Strategy)

我们选定的战场是：**基本面动量 (Fundamental Momentum)**。

正式放弃高频交易（HFT）的军备竞赛，转向 **基本面量化 (Quantamental)**。

• **差异化竞争**：利用个人投资者的**灵活性（空仓权）****敏捷性**，对抗机构的规模劣势。

• **核心逻辑**：。

**Alpha 公式**：

>**Alpha = 优质因子 (数学) + 深度洞察 (AI) - 动态风控 (RL)**`

**差异化优势：**

- **不对称优势 (The Edge)**:
    
    - **灵活性 (Flexibility)**: 利用个人的“空仓权”和“小资金优势”，在机构转身困难的缝隙中寻找 Alpha。
        
    - **技术红利 (Tech Leverage)**: 利用 AI Agent 打破信息不对称，用 XAI 解决信任不对称。
        
- **核心打法**:
    
    - 不赌单次输赢，赌**长期概率**。
        
    - 不看单纯涨幅，看**风险调整后收益**。
        

---

### 4.1 核心引擎：工业级因子工厂 (The Engine)

v2.0 的底层不再是粗糙的脚本，而是标准化的工业流水线。

#### 4.1.1 数据基建：清洗即生命

- **数据清洗 (Hygiene)**: 严格执行 **后复权 (Backward Adjustment)**。
    
    - _痛点_: 只要数据源没处理分红拆股，动量计算就是错误的。
        
    - _标准_: 所有价格计算基于 `Adj Close`。
        
- **存储**: SQLite (热数据) + Parquet (冷数据/回测数据)。
    

#### 4.1.2 核心因子：波动率调整动量 (Sharpe-Momentum)

- **逻辑升级**: 从“谁涨得多”升级为“谁涨得稳”。
    
- **公式**: `Score = Annualized_Return / Annualized_Volatility`。
    
- **效果**: 自动剔除那些暴涨暴跌的“妖股”，筛选出稳健爬升的“白马股”。
    

#### 4.1.3 风险管理：波动率倒数定仓 (Inverse-Vol Sizing)

- **逻辑升级**: 从“固定仓位”升级为“风险平价”。
    
- **机制**:
    
    - `Target_Weight = Risk_Budget / Stock_Volatility`。
        
    - 波动率翻倍，建议仓位自动减半。
        
    - **结果**: 让账户净值曲线更加平滑，减少回撤痛苦。
        

---

### 4.2 智能层：四大“黑科技”护城河 (The Intelligence)

这是 DAAS v2.0 区别于传统量化软件的核心竞争力。

#### 4.2.1 信任层：可解释 AI (XAI / SHAP)

- **痛点**: “系统让我买，但我不知道为什么，心里发慌。”
    
- **方案**: 集成 **SHAP (SHapley Additive exPlanations)**。
    
- **交付**: 在 Alpha Card 上展示 **“得分归因图”**。
    
    - _Example_: “总分 95。正向贡献：动量强劲 (+20)，低空经济热点 (+15)；负向贡献：近期波动率略高 (-5)。” —— **让黑盒变白盒。**
        

#### 4.2.2 逻辑层：金融知识图谱 (Knowledge Graphs)

- **痛点**: 线性思维，看不懂产业链传导。
    
- **方案**: 构建 **产业链图谱 (Supply Chain KG)**。
    
- **交付**: **结构化 Alpha**。
    
    - _Example_: 监控到“碳酸锂”期货涨价 -> 图谱推理 -> 自动推荐下游“宁德时代”或上游“赣锋锂业”。
        

#### 4.2.3 决策层：强化学习 (RL Agent)

- **痛点**: 静态规则（如跌 5% 止损）太死板。
    
- **方案**: 训练 **RL Execution Agent**。
    
- **交付**: **智能交易员**。
    
    - 它不预测股价涨跌，它只预测**“怎么操作最划算”**。在市场恐慌时，它可能建议“分批减仓”而不是“市价止损”，以减少滑点损失。
        

#### 4.2.4 稳健层：合成数据压力测试 (GANs)

- **痛点**: 历史回测容易过拟合（幸存者偏差）。
    
- **方案**: 利用 **GANs (生成对抗网络)** 生成“合成 K 线”。
    
- **交付**: **极限生存测试**。
    
    - 验证策略在 1000 种“从未发生过、但理论上可能发生”的极端行情（如连续 10 个跌停）下，是否会爆仓。

## 5. 产品形态与交互 (UX Design)

为了获客（User Acquisition），产品必须具备极强的**传播属性**。

### 核心界面： "The Alpha Card" (每日战报卡片)

用户每天早上收到一张图（Card），包含三个层级：

1. **信任层 (Trust)** —— _展示肌肉_
    
    - **RPS 评分**: **98.5** (击败 99% 股票)
        
    - **量化信号**: **口袋支点突破** 🟢
        
2. **解释层 (Explanation)** —— _消除认知摩擦_
    
    - **AI 点评**: “主力资金强势介入，DeepSeek 确认其为‘人形机器人’核心龙头，且昨晚公告业绩超预期 30%。技术面刚突破 20 日线压力。”
        
3. **行动层 (Action)** —— _简单执行_
    
    - **操作建议**: **轻仓试错 (10%)**
        
    - **止损红线**: **15.80 元** (跌破即撤)

## 7. 实施路线图 (Execution Roadmap)

我们保持“小步快跑，沿途下蛋”的节奏，但在技术深度上不做妥协。

### Phase 1: 数据地基与工业级因子 (The Foundation) —— _本周_

- **任务**:
    
    1. 搭建 SQLite/Parquet 数据架构。
        
    2. 实现 **后复权 (Adj Close)** 数据清洗管线。
        
    3. 实现 **波动率调整动量** 和 **波动率倒数定仓** 算法。
        
- **交付物**: 一个能跑出“稳健排名”的 Python 引擎。
    

### Phase 2: 可解释的智能 (The Explanation) —— _下周_

- **任务**:
    
    1. 接入 DeepSeek API。
        
    2. 集成 **SHAP 库**，打通“因子 -> 评分 -> 解释”的链路。
        
    3. 生成第一张带有 SHAP 归因的 MVP Alpha Card。
        
- **交付物**: “看得懂、敢跟投”的每日战报。
    

### Phase 3: 黑科技深水区 (The Future) —— _未来规划_

- **任务**:
    
    1. 构建 Neo4j 产业链图谱。
        
    2. 尝试训练简易版 RL Agent (Q-Learning) 用于仓位控制。
        
    3. 引入 GANs 生成数据进行压力测试。
        
- **交付物**: 真正的机构级 AI 投资系统。
